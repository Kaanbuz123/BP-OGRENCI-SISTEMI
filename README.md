# BP-OGRENCI-SISTEMI
Merhabalar Ben Bursa Uludağ Üniversitesi Karacabey Meslek Yüksek Okulu bilgisayar programcılığı 1. sınıf öğrencisiyim.

**BU PROJENİN AMACI:** Karacabey Meslek Yüksekokulu öğrencileri için geliştirilmiş, gerçek zamanlı bir sosyal etkileşim ve kaynak paylaşım portalı.

**KULLANILAN TEKNOLOJİLER:** Next.js, Supabase Database, Tailwind CSS ve Git.

Projenin kaynak kodları, siber güvenlik ve veri gizliliği politikaları gereği özel (private) bir depoda tutulmaktadır.
